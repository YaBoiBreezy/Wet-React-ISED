## Button Tags

Normally components will render a HTML `<Button>` element.
However you can render whatever you'd like.

_For example:_

- Adding a href prop will automatically render an `<a>` element.
- You can use the `as` prop to render whatever your heart desires.

React Bootstrap will take care of the proper ARIA roles for you.

<Canvas>
  <Button variant="primary" href="#">
    Link
  </Button>
  <Button type="submit">Submit</Button>
  <Button as="input" type="button" value="Input" />
  <Button as="input" type="submit" value="Submit"></Button>
  <Button as="input" type="reset" value="Reset" />
</Canvas>
