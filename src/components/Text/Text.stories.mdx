# Text

A component to style text easily. You can modify things like:

- color
- background color
- highlighting
- bold
- italic
- etc.

## Text style

You can also set the text styling. Most standard stylings are supported.

**Please Note:**

- **Mark:** Use to highlight a section of text for reference purposes, due to its relevance in another context.
- **Delete:** Use to indicate text was deleted.
- **Strikethrough:** Use to indicate text that is no longer relevant.
- **Code:** Use to emulate source code and display it in monospace font.
- **Keyboard:** Use to mimic the keys of a computer keyboard.
- **Strong:** Emphasize text by making it bolder
- **Italic:** Emphasize text by italicising

## Text sizing

You can override the global default font-size of `16px`, with a line-height of `1.428`.

**Please note:**

- `h1-h6` modify the size of the text. This is done only through styling. It will not effect page structure.
- `small` is used to define small text, for example side-comments, disclaimers and "fine" print, including copyright and legal text.
- `lead` Make a paragraph stand out by adding setting this to true
