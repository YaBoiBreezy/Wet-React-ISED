# Text

## Text style

You can also set the text styling. Most standard stylings are supported.

**Please Note:**

- **Mark:** Use to highlight a section of text for reference purposes, due to its relevance in another context.
- **Delete:** Use to indicate text was deleted.
- **Strikethrough:** Use to indicate text that is no longer relevant.
- **Code:** Use to emulate source code and display it in monospace font.
- **Keyboard:** Use to mimic the keys of a computer keyboard.
- **Strong:** Emphasize text by making it bolder
- **Italic:** Emphasize text by italicising
