# Title

Generate headings to structure a webpage. You can select the heading levels `h1-h6` by using the `level` prop. It will auto generate html tags `<h1>`to`<h6>`.

## Available Props

### Sample:

<Story name="Title" args={{ level: "h1", children: "Title Example" }}>
  {Template.bind({})}
</Story>

<ArgsTable story="Title" />

## Heading Levels

You can choose form `1-6` levels of headings. The default is level `1`.
